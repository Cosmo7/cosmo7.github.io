For iOS

iOS Mail

Start by adding an account

  1. On your phone, launch Settings
  2. Select Mail, Contacts, Calendars
  3. Under accounts, select Add Account
  4. Choose Other
  5. Under Mail, select Add Mail Account

Account

  • Name: (whatever you like)
  • Email: (your email address)
  • Password: (your password)
  • Description: (whatever you like)

Check you entered the correct details, then click Next

Add account details

Incoming Mail Server

  • Host Name: mail.defco.xyz
  • User Name: (your email address)
  • Password: (your password should already be here)

Outgoing Mail Server

  • Host Name: mail.defco.xyz
  • User Name: (your email address)
  • Password: (your password)

Check you entered the correct details, then click Next

  1. App will take some time verifying and then say “Cannot Connect using SSL”
  2. Click No
  3. Then click Next
  4. App will say “This account may not be able to send or receive emails. Are you sure you want to save?”
  5. Click Save
  6. In the next window, click Save
  7. You should now see your new account in the account list. Click on it.

Change a setting so it works

  1. Click on Account
  2. Scroll down to Advanced and click on it
  3. Scroll down to Incoming Settings
  4. Set Server Port to 143
  5. Click outside the setting so it saves
  6. Click Account to go back
  7. Click Done

Lots of ticks should appear next to settings

Click the home button and start Mail
